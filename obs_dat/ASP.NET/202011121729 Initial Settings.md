# Start

\- initial settings

- install sdk
- open cmd and write

```dotnet
dotnet dev-certs https --trust
```


If it needed to clean certificate use:
```dotnet
dotnet dev-certs https --clean
```


### To create app
Go to the project folder and write
```dotnet
dotnet new webapp
``` 
creating new asp.net core app
