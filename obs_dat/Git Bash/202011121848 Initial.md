==Config setup folder for git bash==
1. Call git bash
2. write: nano ``.bashrc``
3. write
	```git
	write the path
	moveHome() { cd /c/<my/repostiories; } 
	moveHome
	export HOME=/c/my/repositories
   ```

4. Press `CTR+O`
5. Press `CTR+X`
